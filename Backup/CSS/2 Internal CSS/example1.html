Selectors : selector is the concept which represents which properties corresponding to values are going to implement on which element. Types of selectors : 1. Element selector | Tag selector syntax: element_name{ property:value; property:value; | | } example : div{ background-color:red; color:white; } 2. Id Selector (#) syntax : #id_name{ property_name:value; | | } example : #demo{ color:red; }
3. Class Selector (.) syntax : .class_name{ property_name:value; | | } example : .demo{ color:red; }
4. Universal selector (*) syntax : *{ property_name:value; | | } example : *{ color:red; } 5. Grouping selector (,) syntax : element_name1,element_name2,element_name3,.....{ property:value; | | } example : div,span{ color:red; font-size:30px; } 6. Descendant selector syntax : element1 element2 element3 ....{ property:value; | | } example : span{ color:red; } p span{ color:blue; } div p span{ color:purple; }

Direct child selector (>) syntax: element1 > element2{ property:value; | | } example : div > span { color:red; } d. nth-of-type selector :nth-of-type(number | even | odd){ } e. Attribute selector syntax: element_name[type='text']{ } example : input[type='text']{ } f. Pseudo element selector (::) g. Pseudo class selector (:)
